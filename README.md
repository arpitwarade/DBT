## 📌 Project Overview

DBT (Data Build Tool) is used to transform data inside a data warehouse by writing modular SQL queries. It enables developers to apply software engineering practices like version control, testing, and documentation to data pipelines.

---

## 📂 Project Structure

```bash
DBT/
│── models/          # SQL models for transformation
│── macros/          # Reusable SQL logic
│── tests/           # Data quality tests
│── dbt_project.yml  # DBT project configuration
│── packages.yml     # External package dependencies
```

```bash

# Run models

# Test models
```
